# IBM Cloud Full Stack Training Repository

## Overview

Welcome to the **IBM Cloud Full Stack Training Repository**.

This repository contains learning materials, practice projects, assignments, and hands-on examples created during full-stack development training. It covers various technologies used in modern software development, including programming languages, frontend development, backend development, cloud-native technologies, and DevOps practices.

## Technologies Covered

### ☕ Java
- Core Java concepts
- Object-Oriented Programming (OOP)
- Collections Framework
- Exception Handling
- Multithreading
- Java application development practices

### ☕ Spring Boot
- Spring Boot application development
- REST API development
- MVC architecture
- Database integration
- Microservices concepts

### 🌐 JavaScript
- JavaScript fundamentals
- ES6+ features
- DOM manipulation
- Async programming
- Backend development using Node.js

### 🎨 HTML & CSS
- HTML structure and semantic elements
- CSS styling
- Responsive web design
- Frontend UI development

### 🐳 Docker
- Containerization concepts
- Creating Docker images
- Writing Dockerfiles
- Running and managing containers
- Docker-based application deployment

### ☸️ Kubernetes
- Kubernetes architecture
- Pods and deployments
- Services
- Configuration management
- Container orchestration concepts

### 🔄 CI/CD
- Continuous Integration and Continuous Deployment concepts
- Build automation
- Version control workflows
- Deployment pipelines
- DevOps practices
